Para correr el programa necesitas:
1) Tener instalado Visual Studio Code con versión mínima de Python 3.9+
3) Descargar uno o los dos archivos python y el visualizador .html disponibles en este repositorio.
4) Guardar los archivos en una misma carpeta.
5) Abrir VSC y dentro del programa abrir la carpeta donde anteriormente guardamos los archivos.
6) Abrir la Terminal (Ctrl + ñ) y escribir el siguiente comando para instalar las librerías: pip install networkx scikit-learn numpy
7) Ejecutar programa (Run Python File).
